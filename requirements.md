# UniTime Project Requirements

## Project Description
UniTime is a university timetabling system used to manage academic scheduling in universities.  
It helps organize courses, rooms, instructors, and student schedules efficiently.

## Main Requirements
- Manage course scheduling
- Assign instructors to classes
- Allocate rooms for lectures
- Detect scheduling conflicts
- Support timetable generation and updates
- Maintain accurate academic scheduling data

## Project Goal
The goal of this project is to maintain and improve the UniTime backend by applying software maintenance practices on the Java backend.
